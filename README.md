# master-bit-fantasy-console
  
Master-Bit fantasy console software. Use lua files as cartridges.

## Compiling
Code::Blocks project file is provided.  
Requires SDL2, SDL2\_image, and LuaBinaries lua53 libraries.

## Running
Requires SDL2, SDL2\_image, libpng, zlib, and LuaBinaries lua53 binaries.  
Place test.lua and jfont2.png in the program directory.  
Pass cartridge filename to Master-Bit as command line argument. Otherwise, loads "test.lua".
